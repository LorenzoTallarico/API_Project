typedef struct Route Route

Represents a route in the transportation network.

typedef struct Spazzatura Spazzatura

◆ Stazione

typedef struct Stazione Stazione

Represents a station in the transportation network.

This structure contains information about the station, including its unique identifier, relationships with other stations, a binary tree of vehicles, and routing information.

typedef struct Veicolo Veicolo

Represents a vehicle in the transportation system.

This structure contains information about the vehicle's autonomy and pointers to its left and right children in a binary tree.
